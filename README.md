# Lógica de Programação - Visualg

Este repositório contém exercícios e algoritmos desenvolvidos durante meus estudos de lógica de programação.

Tecnologias utilizadas:
- Portugol
- Visualg

Exemplos de exercícios:

- Cálculo de média de alunos
- Comparação entre números
- Simulação de resultado de partidas

Objetivo: praticar estruturas condicionais, variáveis e lógica de programação enquanto estudo Análise e Desenvolvimento de Sistemas.
